# プロジェクト概要: 株式会社メイクアップ コーポレートサイトリニューアル

## 1. プロジェクト目的
株式会社メイクアップのコーポレートサイト（旧: prime-yi.jp）をリニューアルし、以下の4つの事業領域を明確にアピールする「プレミアムでモダン」なWebサイトを構築する。
1.  **WEBシステム開発**
2.  **営業代理**
3.  **コンサルティング**
4.  **コンテンツ制作**

## 2. 技術スタック
-   **Framework**: Next.js 14 (App Router)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS v4
-   **Animation**: Framer Motion
-   **Icons**: Lucide React
-   **Deployment**: Vercel (想定)
