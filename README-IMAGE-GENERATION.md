# makeup-hp サービス画像生成ガイド

## 概要

このスクリプトは、makeup-hpのサービスページ用の4つの画像を自動生成します。hokage-hpプロジェクトの画像生成スクリプトを参考に作成されています。

### 生成される画像

1. **営業代行** (`hero-sales-jp.jpg`)
   - クライアントへの商品プレゼンテーションシーン
   - タブレット/PCでのデータ表示
   - 明確な営業活動を示す構図

2. **コンサルティング** (`hero-consulting-jp.jpg`)
   - 現場での実践的なコンサルティング
   - 工場や職場でのオンサイト業務改善
   - 「手を動かす」実務的なアプローチ

3. **WEBシステム開発** (`hero-dev-jp.jpg`)
   - システム開発とコラボレーション
   - コード、API、システムアーキテクチャ
   - モダンな技術スタック

4. **コンテンツ制作** (`hero-content-jp.jpg`)
   - マルチメディアコンテンツ制作
   - 動画編集、デザイン、撮影機材
   - クリエイティブな制作環境

## セットアップ

### 1. 依存パッケージのインストール

```bash
npm install
```

### 2. 環境変数の設定

`.env` ファイルに以下を追加してください：

```
GEMINI_API_KEY=your_actual_api_key_here
```

**重要**: Google Gemini API キーが必要です。[Google AI Studio](https://makersuite.google.com/app/apikey)で取得できます。

## 画像生成の実行

### 方法1: npm スクリプトを使用

```bash
npm run generate:images
```

### 方法2: 直接実行

```bash
node generate-service-images.mjs
```

## 出力

生成された画像は `docs/images/` ディレクトリに保存されます：

```
docs/images/
├── hero-sales-jp.jpg
├── hero-consulting-jp.jpg
├── hero-dev-jp.jpg
└── hero-content-jp.jpg
```

## プロンプトの特徴

### 営業代行 (Sales)
- **シーン**: 会議室でのクライアントプレゼンテーション
- **強調点**: 実際の営業活動、データ表示、エンゲージメント
- **差別化**: 単なる会議ではなく、明確な商品説明・提案

### コンサルティング (Consulting)
- **シーン**: 現場（工場・職場）でのオンサイトコンサルティング
- **強調点**: 実践的な改善活動、プロセス分析、協働作業
- **差別化**: 会議室ではなく、実際の作業現場での実務的アプローチ

### WEBシステム開発 (Development)
- **シーン**: モダンな開発オフィス
- **強調点**: コーディング、システム設計、技術コラボレーション
- **特徴**: 画面に実際のコードや設計図が表示されている

### コンテンツ制作 (Content)
- **シーン**: クリエイティブスタジオ
- **強調点**: 動画・デザイン制作、撮影機材、マルチメディア
- **特徴**: 制作ソフトウェアと機材が明確に表示されている

## 技術仕様

- **API**: Google Gemini 3 Pro Image Preview
- **エンドポイント**: `https://generativelanguage.googleapis.com/v1beta/models/gemini-3-pro-image-preview:generateContent`
- **画像比率**: 16:9
- **スタイル**: フォトリアリスティック
- **Rate Limiting**: 各生成の間に3秒の待機時間

## トラブルシューティング

### API Key エラー

```
エラー: GEMINI_API_KEY が .env ファイルに設定されていません
```

→ `.env` ファイルに `GEMINI_API_KEY=your_key` を追加してください

### 画像生成エラー

```
API エラー (404): ...
```

→ Gemini API のモデル名や利用可能性を確認してください

### レスポンスに画像データが含まれない

→ プロンプトがポリシーに違反していないか確認してください

## 注意事項

- **課金**: Google Gemini API の利用には課金が発生する場合があります
- **Rate Limiting**: 連続生成時は自動的に3秒間隔を設定しています
- **生成結果**: AI生成のため、毎回異なる結果になります
- **再生成**: 満足のいく結果が得られない場合は再実行してください
